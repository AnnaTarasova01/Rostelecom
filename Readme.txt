Автотесты для формы авторизации Ростелеком.

 Необходим файл с Вашими данными 
	valid_password - пароль
	valid_phone - номер телефона
	valid_email - почта

Содержаться в коллекции auto_test такие тесты как:
- Успешный вход по корректным данным.
- Ошибка при вводе некорректного логина. Проверяет появляется ли сообщение с ошибкой
- Ошибка при вводе некорректного пароля. Проверяет появляется ли сообщение с ошибкой
- Ошибка при оставлении поле логин пустым. Проверяет появляется ли сообщение с ошибкой
- ошибка при сотавлении поля пароль пустым. Проверяет появляется ли сообщение с ошибкой
- Успешный вхо д по временному паролю
- Ошибка при введенном неверном временном пароле. Проверяет появляется ли сообщение с ошибкой
- Ошибка при введении временного кода с истекшим сроком годности. Проверяет появляется ли сообщение с ошибкой
- Тест на всплывающее окно при нажатии на ссылку "помощь"
